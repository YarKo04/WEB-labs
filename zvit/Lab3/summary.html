У цій лабораторній роботі ми досліджували верстку HTML-документів, зокрема блокову верстку та використання CSS і FLEXBOX для створення веб-сторінок. Нашою метою було отримання практичних навичок у верстці сторінок засобами CSS та Flexbox. Під час виконання роботи ми з’ясували переваги та недоліки різних типів макетів веб-сторінок. Вивчення блокової верстки дозволило нам ефективно розміщувати та організовувати елементи сторінок, забезпечуючи їх гнучкість та адаптивність. Застосування CSS дозволило нам створювати стильні та привабливі веб-сторінки. Ми навчилися використовувати різні CSS-властивості для керування виглядом елементів сторінки, включаючи розташування, кольори, шрифти та інші важливі атрибути. Окрім того, ми дослідили принципи FLEXBOX, який дозволяє нам легко створювати гнучкі та адаптивні макети. За допомогою FLEXBOX ми змогли легко управляти розміщенням елементів на сторінці, створювати колонки, вирівнювати елементи та розподіляти простір. В результаті лабораторної роботи ми успішно набули практичних навичок у верстці веб-сторінок за допомогою CSS та FLEXBOX. Ці знання та навички будуть корисні нам при подальшій розробці веб-сайтів і допоможуть нам створювати привабливі та функціональні інтерфейси для користувачів.
